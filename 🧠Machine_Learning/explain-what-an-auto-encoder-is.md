Explain what an <b>Auto-Encoder</b> is.

### Back
An <b>Auto-Encoder</b> is a <b>feed-forward neural net</b> whose job it is to take an input.

To make this non-trivial, we need to add a bottleneck layer whose dimension is much smaller than the input.
