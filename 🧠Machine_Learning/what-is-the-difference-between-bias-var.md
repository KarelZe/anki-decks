### Front
What is the difference between <b>bias</b>, <b>variance</b> and
<b>noise</b>?

### Back
<div>
  <div>
    <div>
      <b>Bias:</b>
    </div>
    <ul>
      <li>Due to restriction of your model class
      <li>Also called “structure error”
    </ul>
    <div>
      <b>Variance</b>:
    </div>
    <ul>
      <li>Due to randomness of the data set
    </ul>
    <div>
      <b>Noise</b>:
    </div>
    <div>
      <ul>
        <li>Nothing we can do about
      </ul>
    </div>
  </div>
