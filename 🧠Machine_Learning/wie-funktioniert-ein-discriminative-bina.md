### Back
<p>Given the training data \(\left(\boldsymbol{x}_{i},
y_{i}\right), \mathrm{i}=1 \ldots \mathrm{N}\), with
\(\boldsymbol{x}_{i} \in \mathbb{R}^{d}\) and \(y_{i} \in\{0,1\}\),
learn a classifier \(f(\boldsymbol{x})\) such that: \[
f\left(\boldsymbol{x}_{i}\right)= \begin{cases}>0, & \text { if
} y_{i}=1 \\ <0, & \text { if } y_{i}=0\end{cases} \]
