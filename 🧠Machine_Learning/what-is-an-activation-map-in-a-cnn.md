### Front
What is an <b>activation map</b> in a CNN?

### Back
<div>An <b>activation map</b> is the output activations for a given <b>filter </b>in a <b>convolutional layer</b>.</div><div>
</div><div>If different filters are applied one can get different activation maps for the same input data.</div><div>
