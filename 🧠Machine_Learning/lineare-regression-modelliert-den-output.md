### Front
<p>Lineare Regression modelliert den Output \(y\) als lineare
Funktion zwischen Input \(x_i\)folgendermaßen:
<p>\[y=f(x)+\epsilon=w_{0}+w_{1} x+\epsilon\]
<p>Was ist dabei die Bedeutung von \(w_0\) und von \(w_1\)? Wie
wirken sich beide geometrisch aus?

### Back

<p>\(w_1\) verändert Steigung der Geraden
