<p>Wie ist das <b>F-Measure</b> (F1 score) definiert?

### Back
<p>\[\text { f-Measure: } \frac{2 \times \text { recall } \times \text { precision }}{\text { recall }+\text { precision }}=\frac{2 \times T P}{2 \times T P+F P+F N}\]
</p>
