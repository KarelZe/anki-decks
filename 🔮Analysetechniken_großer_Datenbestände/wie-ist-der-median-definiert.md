<p>Wie ist der <b>Median</b> definiert?

### Back
<p><b>Definition:</b>
<p>\(x_{m e d}= \begin{cases}x \frac{x_{n+1}}{2} & \text { für } n
\text { ungerade, } \\ \frac{1}{2}\left(x_{n / 2}+x_{n /
2+1}\right) & \text { für } n \text { gerade. }\end{cases}\)
<p><b>Intuition:</b>
<p>Mittlerer Wert, d. h. bei sortierter Anordnung aller Werte der
in der Mitte bzw. Durchschnitt beider Werte in der Mitte bei
gerader Anzahl.
<p>Anwendbar auf <b>numerischen</b> und <b>ordinalen Daten</b>.
<p>Robust bei Ausreisern.
