### Front
Welcher Link besteht zwischen <b>Data Governance</b> und <b>Geschäftsprozessen</b>?

### Back
<ul><li>Prozesslandschaft meint <strong>Unternehmensprozesse</strong>. </li><li>Sichtweise auf Daten in Prozessen wird häufig vernachlässigt.</li><li>Verweben von Daten mit Prozessen durch sg. <strong>Prozess Turtle</strong>. Bei <strong>Prozess Turtle</strong> ist Prozess in Phasen geteilt, Phasen sind durch Gate abgetrennt. Zwischen einzelnen Phasen besteht ein Fluss. Prozess ist erweitert um technische Ressourcen: z. B. EDV-Ausstattung, Prozess-Eingaben, Prozess-Ergebnissen. Prozess-Turtle muss so erweitert werden, dass Verarbeitung von Daten berücksichtigt wird z. B. Löschung von Personenbezogenen Daten, Berechtigungskonzept.</li><li><strong>Beispiel für Turtle Produktentwicklung</strong>: In autonomer Fabrik würde Track-and-Trace Sensor getrackt werden, ob z. B. Gabelstapelfahrer ein Päuschen macht. Prozess-Turtle muss so erweitert werden, dass Verarbeitung von Personenbezogenen Daten berücksichtigt wird.</li></ul>
