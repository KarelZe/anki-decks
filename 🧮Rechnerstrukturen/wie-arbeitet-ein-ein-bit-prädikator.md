### Front
Wie arbeitet ein <b>Ein-Bit Prädikator</b>?

### Back
Ein Ein-Bit-Prädikator hat ein Bit (Vorhersagebit) für die
Kodierung der Vorhersage.
<div>
  Es gibt zwei Zustände:
</div>
<div>
  <div>
    <ul>
      <li>Wenn Bit gesetzt, wird angenommen, dass Sprung ausgeführt
      wird.
      <li>Wenn Bit nicht gesetzt, wird angenommen, dass der Sprung
      nicht ausgeführt wird.
    </ul>
    <div>
      Bei bereits einer Fehlannahme wird Bit invertiert.
    </div>
