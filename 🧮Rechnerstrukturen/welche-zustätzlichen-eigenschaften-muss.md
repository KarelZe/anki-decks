### Front
Welche zustätzlichen Eigenschaften muss eine Pipeline erfüllen damit ein Anhalten der Pipeline komplett umgangen werden kann?

### Back
<ul>
  <li><b>Load-Forwarding:</b> Daten können aus dem Memory oder
  Cache direkt einer ALU zur Verfügung gestellt werden, ohne vorher
  in ein Register geschrieben zu werden.
  <li><b>Result-Forwarding:</b> Das Ergebnis des Operanden kann
  anderen Operanden zur Verfügung gestellt werden noch bevor es in
  das Register zurückgeschrieben ist.
</ul>
