### Front
Was unterscheidet Datenparallelismus von Funktionsparallelismus? Nennen Sie je einen Einsatzschwerpunkt für das jeweilige Konzept.

### Back
<b>Datenparallelismus</b>: gleichzeitiges Ausführen von Operationen
auf getrennten Elementen einer Datenmenge. Einsatzbereich:
Vektorrechner.
<div>
  <div>
    <b>Funktionsparallelismus</b>: gleichzeitiges Ausführen
    mehrerer Operationssammlungen auf (nicht unbedingt getrennten)
    Elementen einer Datenmenge. Einsatzbereich: Prozessorpipeline.
  </div>
</div>
