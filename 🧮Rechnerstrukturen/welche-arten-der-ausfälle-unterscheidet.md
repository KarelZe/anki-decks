### Front
Welche <b>Arten der Ausfälle</b> unterscheidet man?

### Back
<div>
  <div>
    <ul>
      <li><strong>Teilausfall</strong>: Von einer fehlerhaften
      Komponente fallen eine oder mehrere, aber nicht alle
      Funktionen aus.
      <li><strong>Unterlassungsausfall</strong>: Eine fehlerhafte
      Komponente gibt eine Zeit lang keine Ergebnisse aus. Wenn
      jedoch ein Ergebnis ausgegeben wird, dann ist dieses korrekt.
      <li><strong>Anhalteausfall</strong>: Eine fehlerhafte
      Komponente gibt nie mehr ein Ergebnis aus.
      <li><strong>Haftausfall</strong>: Eine fehlerhafte Komponente
      gibt ständig den gleichen Ergebniswert aus.
      <li><strong>Binärstellenausfall</strong>: Ein Fehler
      verfälscht eine oder mehrere Binärstellen des Ergebnisses.
    </ul>
  </div>
