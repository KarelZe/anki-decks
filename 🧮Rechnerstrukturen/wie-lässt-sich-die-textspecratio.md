Wie lässt sich die \(\text{SPECratio}\) zum Vergleich zweier Rechner verwenden?

### Back
\[\frac{\text { SPECratio } A}{\text { SPECratio } B}=\frac{\frac{\text { Referenzzeit }}{\text { Ausführungszeit A }}}{\frac{\text { Referenzzeit }}{\text { Ausführungszeit B }}}=\frac{\text { Ausführungszeit B }}{\text { Ausführungszeit A }}\]
