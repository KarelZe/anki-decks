## Front
What is a <b>conjugate prior</b>?

## Back
If the <b>posterior distribution</b> \(p(\boldsymbol{\theta} \mid
\mathcal{D})\) ist the same <b>probability distribution family</b>
as the prior probabilty distribution \(p(\theta)\) the prior and
posterior are then called <b>conjugate distributions</b>.
