## Front
Was zeichnet einen <b>idealen Entscheidungsträger</b> im Kontext
von <b>BibTip</b> aus?

## Back
<ul>
  <li>BibTip basiert auf idealem Entscheidungsträger ohne
  Präferenzen.
  <li>Macht Entscheidungen zufällig.
  <li>Basiert auf dem Konzept des <em>choice sets</em>, aus dem er
  auswählt. Dieses teilt sich:
  <li>\(\text { Total set } \supset \text { awareness set }
  \supseteq \text { consideration set } \supseteq \text { choice
  set }\)
    <ul>
      <li><em>Total Set</em> = Alle in Volkswirtschaft verfügbare
      Produkte
      <li><em>Awareness Set</em> = Alle mir bekannte Produkte
      <li><em>Consideration Set</em> = Alle Produkte, die ich mir
      überlege zu kaufen.
      <li><em>Choice Set</em> = Was im Laden vorhanden ist.
    </ul>
</ul>
