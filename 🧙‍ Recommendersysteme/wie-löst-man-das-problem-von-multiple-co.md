Wie löst man das Problem von <b>Multiple Components</b>?

### Back
Page-Rank löst das Problem mehrerer nicht-verbundener Komponenten, indem es eine Matrix \(M\) aufbaut, die der gewichteten Summe der Matrix \(A\), die aus der Linkstruktur abgeleitet ist und einer Matrix \(S\), welches eine Gleichverteilung von Webseiten enthält, aufbaut.
