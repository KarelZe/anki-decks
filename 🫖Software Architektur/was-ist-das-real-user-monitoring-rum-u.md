### Front
Was ist das <b>Real User Monitoring (RUM)</b> und was ist sein <b>Ziel</b>?

### Back
<ul>
  <li>Leichtgewichtige Technik
  <li>Gerne bei Webapplikationen verwendet
  <li>Gibt Einblicke in die Anwendungsperformanz aus
  Nutzerperspektive
  <li>Stellt eine nicht-aufdringliche Instrumentierung der Dienste
  einer Anwendung bereit
  <li>Angewendet während des Betriebs
</ul>
<div>
  <strong>Ziel</strong>
</div>
<ul>
  <li>Nutzerverhalten verstehen z. B. häufig genutzte Features
  <li>Die Servicequalität inkl. Performanz und Verlässlichkeit zu
  erreichen.
</ul>
<div>
  <strong>Umsetzung</strong>
</div>
<ul>
  <li>Einbindung von Java-Skript-Snippets
</ul>
<div>
  <strong>Output</strong>
</div>
<ul>
  <li>Information über Nutzerverhalten, User-Experience z. B. was
  mag er?
</ul>
