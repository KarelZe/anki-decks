### Front
Was zeichnet eine <b>Client-Server-Architektur</b> aus?

### Back
<ul>
<li>Verteiltes System Model (<em>Distributed System Model</em>), das zeigt, wie Daten verarbeitet werden über eine Vielzahl von Komponenten</li>
<li>Eine Menge unabhängiger Server, die spezfische Dienste <em>(Services)</em> wie Druck, Datenmanagement o. Ä. anbieten</li>
<li>Menge an Clients, die bestimmte Dienste aufrufen</li>
<li>Netzwerk, dass Clients erlaubt, auf Server zuzugreifen</li>
</ul>
