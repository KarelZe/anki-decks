### Front
Welche Funktion leistet das <b>Component Repository</b> für den
<b>Software-Architekt</b>, welche für den <b>Implementierer</b>?

### Back
<div><b>Software-Architekt:</b></div><ul><li>frägt Komponenten ab</li>
<li>Spezifiziert neue Komponenten </li><li>abstrahiert Komponenten anhand des Repositories</li>
</ul><div>D. h. Software-Architekt nimmt bestehende Komponenten und kombiniert sie zu einem neuen System. Benötigt eine Software neue Komponenten, dann können sie erzeugt werden.</div><div></div><div><b>Komponenten-Entwickler:</b></div>
<ul>
<li>Erstellt konkrete Komponenten</li><li>Legt konkrete Komponenten ab</li>
</ul><div>Der Entwickler hingegen kann abstrakte Komponenten als Vorlage für konkrete Komponenten nehmen. Oder Entwickler kann bestehende Komponenten nehmen und diese zu einer neuen Komponente  komponieren.</div>
