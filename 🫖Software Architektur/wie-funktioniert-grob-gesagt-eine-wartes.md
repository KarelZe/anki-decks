### Back
<div>Eine <b>Warteschlange </b><i>(queue)</i> besteht aus  einer Warte-Schlange (<i>waiting line</i>) und einem oder mehrerer Server. Jobs, welche die Warteschlange erreichen werden entweder sofort bedient, sofern min. einer der Server frei ist oder in die Warteschlange eingereiht.</div><div>
</div><div>Server arbeitet die Jobs ab. </div><div>
</div><div>Es gibt jeweils eine Verteilungsfunktion wie lange Server braucht, den Job abzuarbeiten und wie viele Jobs ankommen. </div><div>
