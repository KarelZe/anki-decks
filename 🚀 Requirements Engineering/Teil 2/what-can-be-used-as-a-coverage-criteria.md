What can be used as a <b>coverage criteria</b>?

### Back
<ul>
  <li><b>Requirements coverage:</b> At least one case per
  requirement
  <li><b>Function coverage:</b> At least one case per function
  <li><b>Scenario coverage:</b> For every type scenario / use case
  with all actions and all branches covered
</ul>
