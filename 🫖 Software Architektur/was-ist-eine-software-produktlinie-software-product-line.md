## Back
<div>
  Eine Software Produktlinie ist eine Menge von Software-lastigen
  Systemen, die eine gemeinsame Kern an Features teilen, die
  bestimmte Bedürfnisse eines Marktsegments oder einer Mission
  erfüllen und aus einer gemeinsamen Menge an zentralen Assets in
  vorgeschriebener Art und Weise entwickelt werden.
