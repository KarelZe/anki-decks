### Back
<div>
  <strong>Komponente</strong>
</div>
<ul>
  <li>Stellt Funktionalität bereit
  <li>Allgemein, Wiederverwendbar, durch Dritte
  <li>Bausteine für fertiges System
  <li>Verteilt oder lokal
</ul>
<div>
  <strong>Konnektoren</strong>
</div>
<ul>
  <li>Verbinden Komponenten
  <li>Oftmals spezialisiert für das System
  <li>Erzeugt zur Deployment-Zeit
</ul>
