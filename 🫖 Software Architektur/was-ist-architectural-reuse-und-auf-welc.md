### Front
Was ist <b>Architectural Reuse</b> und auf welchen Ebenen kann
dieser erfolgen?

### Back
<div>
  Wiederverwendung von Komponenten bezieht sich typischerweise auf
  die Wiederverwendung der Implementierung von Komponenten.
  <div>
    <b>Klassifizierung nach Design- bzw. Codeorientierung</b>
    <div><img src= 
    "paste-608053465523f7178d856c94584e7a708753fb94.jpg"></div>
  </div>
</div>
<div>
  <b>Code-Ebene:</b> Extrem-Beispiel Code-Schnipsel von Stack
  Overflow. Erfordert Anpassungen z. B. Variablen. Man hat keine
  Kapselung.
</div>
<div>
  <b>Interfaces:</b> Zwischendrin, weil in Code ausdrückbar. Welche
  Interfaces man erzeugt, hängt aber von Entwurf ab.
</div>
<div>
  <b>Design-Ebene:</b> Wiederverwendung von Entwurfswissen. Nicht
  den Code.
</div>
