## Front
Wie unterscheidet sich ein <b>Framework</b> von einer
<b>Library</b>?

## Back
<b>Framework</b>: "main" ist vorhanden. Programmierer füllt noch
noch Variationspunkte im Code, die vom Framework aufgerufen werden.
<div>
  <b>Library</b>: Andersherum. Man schreibt die "main" und wählt
  dann aus Library aus, was aufgerufen wird z. B. Mathe-Bibliothek.
</div>
