## Front
Give the <b>definition</b> of the <b>Soft max function</b>.

## Back
Probability of class \(k\) given input image \(\boldsymbol{x}\) (soft-max)
\[
p\left(c_{k}=1 \mid x\right)=\frac{e^{o_{k}}}{\sum_{j} e^{o_{j}}}
\]
