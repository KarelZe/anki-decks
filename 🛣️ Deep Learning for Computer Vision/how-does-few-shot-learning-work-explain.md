### Back
Few-Shot Learning in a \(n\)-way \(k\)-shot Scenario.
<b>Training:</b> Large set of image, segmentation-mask pairs
\(\left(x^{\text {train }}, m^{\text {train }}\right)\) for some
training classes. <b>Testing:</b> Support set of \(k\) image,
segmentation-mask pairs for \(n\) novel classes. <b>Goal:</b>
Segment test images into the \(n\) novel classes.
