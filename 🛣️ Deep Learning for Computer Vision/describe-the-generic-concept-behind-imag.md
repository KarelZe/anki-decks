### Front
Describe the generic concept behind image segmentation through clustering.

### Back
Process of partitioning a set of patterns into <b>clusters</b>. Find subsets of points which are close together.

Cluster pixels based in intensity values, color properties, motion / optical flow properties, texture measurements etc.
