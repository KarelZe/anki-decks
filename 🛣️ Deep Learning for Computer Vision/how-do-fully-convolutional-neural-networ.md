### Back
<ul>
  <li>Convolutional and pooling layers followed by upsampling
  layers
  <li>The output layer has a dimension of Height x Width x #Classes
  <li>Each filter \(c\) in the output layer shows the probability
  that a pixel belongs to the class \(c\)
