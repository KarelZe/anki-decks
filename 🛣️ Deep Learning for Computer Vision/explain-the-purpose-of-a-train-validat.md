Explain the purpose of a <b>train / validation</b> and <b>test
set</b> in <b>machine learning</b>.

### Back
<b>Training set</b>
<ul>
  <li>A training data set is a data set of examples used during the
  learning process and for fitting the parameters e. g., weights.
  <li>In supervised deep learning annotations
</ul><b>Test set</b>
<ul>
  <li>A test data set is a data set, independent from the training
  set, that follows a similar probability distribution as the
  training set.
</ul><b>Validation set / Dev set</b>
<ul>
  <li>dataset used to tune the hyperparameters.
</ul>
