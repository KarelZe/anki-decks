## Front
How is the <b>pixel accuracy</b> defined?

## Back
\[
\text { accuracy }=\frac{T P+T N}{T P+T N+F P+F N}
\]
\(TP\) correctly classified pixel for given class \(TN\) correctly classified as not belonging to a given class.
