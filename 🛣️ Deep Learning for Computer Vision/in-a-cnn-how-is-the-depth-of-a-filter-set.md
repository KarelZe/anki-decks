## Front
In a <b>CNN</b> how is the <b>depth</b> of a <b>filter</b> set?

## Back
Filters always extend to the <b>full depth</b> of the <b>input
volume</b>. That is, if an image contains 3 channels, the filter in
a convolutional layer will also have a depth of three.
