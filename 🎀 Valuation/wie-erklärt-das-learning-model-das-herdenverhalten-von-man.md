## Front
<p><span>Wie erklärt das Learning Model das Herdenverhalten von
Managern?</span>

## Back
<ul style= 
"font-weight:400;letter-spacing:normal;text-indent:0px;text-transform:none;white-space:normal;word-spacing:0px">
  <li>
    <p><span>Es gibt 2 Arten von Managern:</span>
    <ul>
      <li><span>Schlaue, welche Informationssignale über den Wert
      von Investments empfangen</span>
      <li><span>Dumme, die nur rauschende Signale empfangen.</span>
    </ul>
  <li>
    <p><span>Erst nach einer Investment-Entscheidung können Manager
    sich zu einer der beiden zählen.</span>
  <li>
    <p><span>Ist das Investment großteils schlecht
    prognostizierbar, ist es wahrscheinlich, dass man die
    Fehleinschätzung mit anderen Managern teilt.</span>
  <li>
    <p><span>Das Festhalten an einer Investment-Entscheidung wird
    besser aufgenommen, wenn sie nicht konträr zu der anderer
    Manager ist.</span>
  <li>
    <p><span>Ist die Investment-Entscheidung nachteilig, ist dies
    damit nicht so schlimm, sofern andere auch diesen Fehler
    machen.</span>
  <li>
    <p><span>Folgt ein Manager nun also der Herde, wird dies am
    Arbeitsmarkt interpretiert als hätte er das Signal empfangen.
    Andernfalls als dumm.</span>
</ul>
