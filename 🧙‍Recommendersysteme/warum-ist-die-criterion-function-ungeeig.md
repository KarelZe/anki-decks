### Front
Warum ist die Criterion Function ungeeignet, um die <b>optimale
Clusteranzahl</b> zu bestimmen?

### Back
<ul>
  <li>Problematisch an Criterion Function \(J\) ist, dass \(J\)
  minimal ist, sofern die Zahl der Cluster \(c\) der Anzahl der
  Knoten \(n\) entspricht.
  <li>Bis zur optimalen Clusteranzahl \(c\) rasch abnimmt. Danach
  nimmt sie weiter ab, jedoch langsamer. \(J\) ist minimal sofern
  die Zahl der Cluster \(c=n\) entspricht.
</ul>
