<div>
  <b>Input-Dimension</b>
</div>
<ul>
  <li>(Individuelle) Informationen über Empfehlungsempfänger
    <ul>
      <li>Textkommentare
      <li>Bewertungen (Auf Skalen, mit Anreizsystemen (Incentives))
      <li>Schlüsselworte für Sachverhalte (z.B. ergeben
      Interessensprofile)
      <li>Explizite Navigation (Wahlmöglichkeiten werden
      protokolliert, Wahl des Nutzers erforderlich)
      <li>Implizite Navigation (Aus Clickstreams ohne Interaktion
      mit Nutzer)
      <li>Kaufgeschichte (explizit (Absichten/Meinungen) / implizit
      (Verhalten))
      <li>Informationen über andere (potentielle) Nutzer
      <li>Informationen über empfehlenswerte Angebote
    </ul>
  <li>(Aggregierte) Informationen über (virtuelle) Gemeinschaften
    <ul>
      <li>Textkommentare (Editiert / nicht editiert)
      <li>Bewertungen (Aggregiert auf Gruppenniveau)
      <li>Merkmalszuordnungen für Sachverhalte
      (Durchschnittsmeinungen / Auswahl von Merkmalen der
      Gemeinschaft)
      <li>(“Kauf”-)Geschichte der Gemeinschaft
      <li>Externe Zusatzinformation
    </ul>
</ul>
<div>
  <b>Output-Dimension</b>
</div>
<ul>
  <li>Personalisierungsniveau
    <ul>
      <li>Keine Personalisierung
      <li>Aktuelle Personalisierung (z.B. Session / Warenkorb)
      <li>Langfristige Personalisierung (z.B. Kaufgeschichte)
    </ul>
  <li>Art der Informationsübermittlung
    <ul>
      <li>Pull-Technologie (Nutzer holt aktiv Empfehlung)
      <li>Push-Technologie (Empfehlungen werden an Nutzer
      geschickt)
      <li>Nutzungsbegleitende Technologien(Empfehlungen beziehen
      sich z.B. auf aktuelles Navigationsverhalten)
    </ul>
  <li>Informationsaufbereitung
    <ul>
      <li>Bewertungen (Produkt mit z.B. Note)
      <li>Kurzberichte (z.B. Produktkritik)
      <li>Vorhersagen (Produkt und z.B. Wahrscheinlichkeiten)
      <li>Vorschläge (Hinweise z.B. auf verwandte Produkte)
    </ul>
</ul>
